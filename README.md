
# GlobalDock Premium Manual Import

Original marketplace-style Next.js template with white premium UI, manual product import, protected admin, search, categories, and optional GitHub sync.

## Local run
1. Copy `.env.example` to `.env.local`
2. Set `ADMIN_PASSWORD`
3. `npm install`
4. `npm run dev`

## Production note
This demo saves JSON files locally. On Vercel, local file writes are not persistent. For production, use either GitHub sync or a database.
